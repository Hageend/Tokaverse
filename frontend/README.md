# TOKAVERSE
Un hub de Mini Apps gamificadas donde el wallet de Toka es el centro: paga, gana puntos, desbloquea niveles, reta a amigos y canjea premios reales, todo sin salir del ecosistema Toka.
## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```
